# Roadmap — AgroEnergy Insight Builder

Este documento descreve o planejamento estratégico e a evolução do sistema de análise de faturas.

## 🏁 Estado Atual (V1.0)
- Processamento básico de faturas Copel.
- Dashboard de autoconsumo e créditos solares.
- Exportação rudimentar via PDF/Canvas.

## 📈 Marcos de Evolução

### Fase 1: Expansão de Compatibilidade (2024-2025)
- [ ] Suporte para concessionárias adicionais (e.g., CPFL, Enel).
- [ ] Refinamento do parser de PDF para lidar com diferentes layouts de faturas.
- [ ] Implementação de histórico de faturas persistente (Base de dados local ou nuvem).

### Fase 2: Inteligência e Predição (2025-2026)
- [ ] Alertas proativos de fim de créditos com base em sazonalidade.
- [ ] Sugestões de redimensionamento de usina solar com base no burn rate.
- [ ] Integração com APIs climáticas para correlação de geração.

### Fase 3: Ecossistema Agro (2026+)
- [ ] Dashboard multi-unidade para produtores com várias matrículas.
- [ ] Relatórios automáticos de economia de CO2.
- [ ] Interface mobile-first para produtores em campo.
